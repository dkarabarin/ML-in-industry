# Прогнозирование температуры стали в металлургическом производстве
# Описание проекта
Проект посвящен разработке модели машинного обучения для прогнозирования температуры стали на металлургическом комбинате "Стальная птица" с целью оптимизации потребления электроэнергии. На основе исторических данных технологического процесса разработана модель, которая предсказывает конечную температуру сплава с высокой точностью.

# Основные цели исследования:
Анализ параметров технологического процесса выплавки стали
Разработка модели для прогнозирования конечной температуры сплава
Оптимизация энергопотребления за счет точного контроля температуры
Формирование рекомендаций по улучшению технологического процесса

# Стек технологий
Python - основной язык программирования
Pandas & NumPy - обработка и анализ данных
Scikit-learn - библиотека для машинного обучения
LightGBM - градиентный бустинг
Matplotlib/Seaborn - визуализация данных
Phik - расширенный корреляционный анализ

# Ключевые результаты
Качество модели
MAPE (средняя абсолютная процентная ошибка): 0.42%
MAE (средняя абсолютная ошибка): 6.71°C
RMSE (среднеквадратичная ошибка): 9.00°C
Точность предсказания: 99.58%

# Лучшая модель
Алгоритм: Random Forest Regressor
